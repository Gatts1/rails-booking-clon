# Documentation

## How do install

You need install the Gemfile

```
bundle install
```

Now run the migrates and seeds

```
rails db:setup
```

Don't forget add `.env`, you can copy a template of [.env.example](https://github.com/codeableorg/rails-booking-clon/blob/master/.env.example)
